# Core Distinctions: Экосистема развития интеллекта

## Обязательные различения

### D.ECO.001: Экосистема ≠ Набор ресурсов

| Экосистема | Набор ресурсов |
|------------|----------------|
| Связанная целостность | Коллекция элементов |
| Бесшовные переходы | Независимые единицы |
| Общая цель | Разрозненные функции |

**Почему важно**: Набор ресурсов без связей не создаёт экосистему. Экосистема предполагает спроектированные переходы и общую архитектуру.

---

### D.ECO.002: Компонент ≠ Продукт

| Компонент экосистемы | Продукт |
|----------------------|---------|
| Функция в системе | Самостоятельная ценность |
| Связан с другими | Автономен |
| Оценивается по вкладу в целое | Оценивается сам по себе |

**Почему важно**: Компонент экосистемы имеет смысл в контексте целого. Курс как продукт и курс как компонент экосистемы — разные вещи.

---

### D.ECO.003: Бесшовность ≠ Единообразие

| Бесшовность | Единообразие |
|-------------|--------------|
| Отсутствие барьеров | Одинаковый формат |
| Плавные переходы | Один стиль везде |
| Связность опыта | Унификация |

**Почему важно**: Бесшовность достигается не унификацией, а качеством переходов между разными компонентами.

---

### D.ECO.004: Путь ≠ Маршрут обучения

| Путь в экосистеме | Маршрут обучения |
|-------------------|------------------|
| Траектория через компоненты | Последовательность уроков |
| Может быть нелинейным | Линейная структура |
| Определяется состоянием | Определяется программой |

**Почему важно**: Путь в экосистеме адаптивен к состоянию созидателя, а не жёстко предписан.

---

### D.ECO.005: Точка входа ≠ Начальный курс

| Точка входа | Начальный курс |
|-------------|----------------|
| Любой первый контакт | Обучающий продукт |
| Может быть событием, сообществом | Контент |
| Зависит от созидателя | Фиксирован |

**Почему важно**: В экосистеме много точек входа для разных типов созидателей.

---

### D.ECO.006: Качество экосистемы ≠ Качество компонентов

| Качество экосистемы | Качество компонентов |
|---------------------|----------------------|
| Бесшовность целого | Качество частей |
| Связность | Контент |
| Конверсия переходов | Удовлетворённость |

**Почему важно**: Отличные компоненты могут образовывать плохую экосистему, если между ними барьеры.

## Применение различений

При описании экосистемы проверяйте:
- [ ] Не путаете ли экосистему с набором ресурсов?
- [ ] Не путаете ли компонент с продуктом?
- [ ] Не путаете ли бесшовность с единообразием?
- [ ] Различаете ли путь и маршрут?
