# Bounded Context: Экосистема развития интеллекта

## Определение области

**Экосистема развития интеллекта** — связанная совокупность ресурсов, сервисов и сообществ, обеспечивающих бесшовный путь развития от первого знакомства до системного мышления мирового уровня.

## Ключевые понятия

| Термин | Определение |
|--------|-------------|
| **Экосистема** | Целостная система компонентов, связанных для достижения цели развития интеллекта |
| **Бесшовность** | Отсутствие разрывов и барьеров между этапами и компонентами |
| **Компонент экосистемы** | Отдельный ресурс/сервис/сообщество, выполняющий определённую функцию |
| **Путь развития** | Траектория прохождения через компоненты экосистемы |
| **Точка входа** | Место первого контакта с экосистемой |
| **Переход** | Смена одного компонента на другой в рамках пути |

## Границы области

### Что входит

- Типы компонентов экосистемы
- Связи между компонентами
- Критерии бесшовности
- Методы оценки качества экосистемы
- Паттерны построения экосистемы
- Типовые ошибки

### Что НЕ входит

- Конкретное содержание курсов
- Персональное развитие созидателя (см. spf-personal-pack)
- ИТ-реализация (см. spf-digital-platform-pack)
- Методология обучения (downstream)

## Семантические границы

### Экосистема vs Платформа

| Экосистема | Платформа |
|------------|-----------|
| Концептуальная целостность | Техническая реализация |
| Компоненты любого типа | ИТ-сервисы |
| Бесшовность пути | Интеграция систем |

### Экосистема vs Сообщество

| Экосистема | Сообщество |
|------------|------------|
| Включает сообщества как компонент | Один компонент |
| Структурированная целостность | Социальная связность |

## Критерии истины

Утверждение об экосистеме считается **проверяемым**, если:
1. Можно указать конкретные компоненты
2. Можно проверить наличие связей
3. Можно измерить бесшовность перехода
4. Можно наблюдать прохождение пути
