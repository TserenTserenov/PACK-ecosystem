# ECO.SOTA.007: Remote-First Team Building (2025-2026)

## Контекст

Remote-first ≠ remote-friendly. Remote-first = async по умолчанию, sync по необходимости. В 2026 лучшие engineering teams — async-first by design, не by accident.

---

## Async-First принцип

### Sync vs Async: decision rule

| Sync (realtime) | Async (recorded/written) |
|----------------|------------------------|
| Решения с дебатом | Status updates |
| Brainstorming | Feature demos |
| Relationship building | Code reviews |
| Sensitive conversations | Weekly summaries |
| Onboarding (first days) | Documentation |

**Правило:** Перед созданием встречи спросить: «Нужна ли real-time дискуссия?» Если нет → async.

## Communication Stack (SOTA 2026)

| Слой | Инструмент | Cadence |
|------|-----------|---------|
| **Real-time** | Slack/Discord (с norms) | Response: 4h (async channels), 30min (urgent) |
| **Recorded** | Loom / async video | Stand-ups, demos, walkthroughs |
| **Documented** | Notion / Confluence | Decisions, specs, runbooks |
| **Task tracking** | Linear / GitHub Projects | Continuous |
| **Deep work** | Focus blocks в календаре | Daily, 2-4h protected |

### Slack/Discord norms

- **#urgent** — response <30 min (emergencies only)
- **#general** — response <4h (business hours)
- **#random** — optional, social
- **Threads always** — no messages in main channel
- **No expectation of instant reply** — если срочно, перейти в #urgent

## Meeting Rituals (минимальный набор)

| Ritual | Частота | Формат | Длительность |
|--------|---------|--------|-------------|
| **Weekly sync** | 1×/неделя | Video | 30 мин |
| **1:1** | 1×/2 недели | Video | 25 мин |
| **Sprint planning** | 1×/2 недели | Video | 45 мин |
| **Retro** | 1×/2 недели | Async doc → 15 мин sync | 15 мин |
| **Social** | 1×/неделя | Optional, informal | 30 мин |

**Правило:** Meetings have agenda (written before), notes (shared after), action items (assigned).

## Culture in Remote-First

### Intentional culture design

> Культура в remote не возникает случайно — её нужно проектировать.

| Элемент | Реализация |
|---------|-----------|
| **Trust by default** | Assume competence, verify by outcomes |
| **Written-first** | Решения документируются, не обсуждаются устно |
| **Timezone respect** | Overlap hours определены, остальное = async |
| **Social glue** | Scheduled informal time, pair programming, virtual coffee |
| **Transparency** | All-hands notes, open channels, public docs |

### Onboarding remote

| День | Фокус |
|------|-------|
| 1 | Access, tools, buddy assignment |
| 2-3 | Codebase walkthrough (recorded + live) |
| 4-5 | First small PR (paired) |
| Week 2 | Independent small task |
| Month 1 | Full sprint participation |

## Ключевые выводы для IWE

1. **Async-first by design:** Документация, async stand-ups (Loom), minimal meetings
2. **AI = force multiplier:** Claude Code + automated testing = lean team output
3. **Written-first culture:** Решения в Notion/GitHub, не в Slack threads
4. **Timezone overlap:** Определить 2-3h overlap для sync, остальное async
5. **Trust by default:** Измерять outcomes (shipped features), не activity (hours)
