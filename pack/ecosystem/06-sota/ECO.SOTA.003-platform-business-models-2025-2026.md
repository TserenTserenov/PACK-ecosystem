# ECO.SOTA.003: Platform Business Models (2025-2026)

## Контекст

К 2026 году более $3 трлн глобального ВВП обеспечивается API-транзакциями (Gartner). Платформенные компании получают 2-3× более высокую оценку, чем standalone-продукты. EdTech рынок: $214B (2026), CAGR 13.45%, AI — обязательный baseline с 2026.

---

## 7 моделей создания экосистемной ценности (EY)

| Модель | Суть | Когда применять | Примеры |
|--------|------|-----------------|---------|
| **Marketplace** | Соединяет поставщиков и потребителей | Фрагментированный рынок, высокие transaction costs | Udemy, Coursera for Business |
| **Orchestrator** | Координирует цепочку создания ценности | Сложный продукт из компонентов разных поставщиков | IWE (методология + LMS + AI + сообщество) |
| **Platform-as-a-Service** | Предоставляет инфраструктуру для строительства | Есть базовые capabilities, которые нужны многим | Moodle, Canvas LMS |
| **Data Ecosystem** | Создаёт ценность из агрегированных данных | Большой объём данных от участников | Digital Twin, learning analytics |
| **Community** | Ценность из взаимодействий между участниками | Экспертное знание, peer learning | Клуб, Stack Overflow |
| **Co-creation** | Участники совместно создают продукт | Пользователи = со-авторы | Open-source LMS, user-generated content |
| **Shared Services** | Общая инфраструктура для группы организаций | Высокая стоимость дублирования | Shared identity (Ory), shared AI |

## EdTech-специфичные тренды 2025-2026

### AI как обязательный baseline

2026 — tipping point: экспериментальные AI-инструменты → обязательный baseline. Durable use cases: course design, assessment support, student services, workforce training.

### Модели монетизации

| Модель | Тренд 2025-2026 | Применимость к IWE |
|--------|-----------------|---------------------|
| **Freemium + AI upsell** | AI-фичи как premium tier | Digital Twin free → AI coaching paid |
| **Usage-based API** | Pay per API call для интеграторов | Agent SDK: per-request pricing |
| **Subscription + marketplace commission** | Platform fee + % от третьих сторон | Подписка + комиссия с partner content |
| **Data insights** | Анонимизированные insights для B2B | Learning analytics для HR/L&D |

### Инвестиционный фокус

- VC в EdTech: $2.4B (2025), рост 20% vs 2024
- AI-powered стартапы: средний seed $7.8M (vs $4M non-AI)
- Приоритеты: AI tutoring, assessment/analytics, workforce upskilling

---

## API Economy тренды

1. **From platforms to ecosystems:** Организации переходят от закрытых платформ к экосистемному подходу с партнёрствами, стандартами и marketplace
2. **API monetization 2.0:** Гибкие модели (usage-based billing, partner revenue sharing, tiered access)
3. **Composable architecture:** Модульные компоненты, собираемые через API (Gartner: «composable business»)
4. **Regulatory compliance as driver:** GDPR, AI Act → стандартизация API и governance

---

## Distributed Network Models (тренд 2026)

Сдвиг от иерархий к распределённым сетям: peer-to-peer marketplace, community platforms, децентрализованное governance. Для IWE: выпускники как авторы контента, менторы как независимые агенты.

---

## Ключевые выводы для IWE

1. **Модель Orchestrator** наиболее релевантна: IWE координирует методологию, AI, LMS и сообщество
2. **AI — не опция, а baseline** с 2026: Digital Twin и AI coaching должны быть core, не add-on
3. **Community-Led Growth** подтверждён как модель для экспертных продуктов (ECO.M.004)
4. **API-first** необходим для перехода от Product к Platform: Agent SDK, Digital Twin API
5. **Freemium + AI upsell** — оптимальная модель монетизации на seed-стадии
