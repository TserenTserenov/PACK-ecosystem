# ECO.D.003: Education ≠ Intelligence Development

## Таблица сравнения

| Критерий | Education (Образование) | Intelligence Development (Развитие интеллекта) |
|----------|------------------------|-----------------------------------------------|
| **Цель** | Передать знания/навыки | Усилить способность мыслить |
| **Продукт** | Курс, диплом, сертификат | Интеллектуальная среда (IWE) |
| **Метрика успеха** | Completion rate, оценка, трудоустройство | Качество решений, switching cost, NRR |
| **Горизонт** | Семестр, курс (конечный) | Пожизненный (среда растёт с участником) |
| **Роль AI** | Тьютор (отвечает на вопросы) | Экзоскелет (усиливает мышление) |
| **Конкуренты** | Coursera, вузы, bootcamps, корп. L&D | Нет прямых (новая категория) |
| **Бизнес-модель** | Продажа контента/доступа | Продажа среды и практики (контент открыт) |
| **Retention driver** | Контент (конечный → churn) | Data moat (ЦД → switching cost растёт) |

## Ключевое различение

Education = передача информации от источника к получателю. Intelligence Development = усиление когнитивных способностей через методологию + среду + практику.

Рынок образования ($350B) — **входная воронка**, не конкурентное поле. Люди приходят за «курсами», остаются за средой. Позиционирование: используем язык образования для привлечения, но продукт — другого класса (Intelligence Platform).

## Failure Mode: Positioning Trap

Если позиционироваться как EdTech:
- Инвесторы сравнивают с Coursera/Stepik → оценка по их мультипликаторам (низкие)
- Пользователи ожидают «курсы» → разочарование от методологической глубины
- Churn benchmarks EdTech (9.6% monthly) применяются как потолок, а не как проблема для решения

Если позиционироваться как Intelligence Platform:
- Новая категория → нет прямых конкурентов → premium valuation
- Пользователи ожидают «среду» → готовы к долгосрочному engagement
- Switching cost (ЦД + Pack) → retention выше рынка

## Связь с рынком образования

Рынок образования нельзя игнорировать:
1. **Воронка:** люди ищут «курсы», «обучение», «развитие» — это язык входа
2. **Benchmarks:** EdTech unit economics (churn, CAC, NRR) — точка сравнения для инвесторов
3. **L&D бюджеты:** корпоративные клиенты тратят из L&D-бюджетов — нужно попадать в эту категорию
4. **Workforce upskilling:** 44.8% CAGR — растущий сегмент, где Intelligence Development = premium tier

**Стратегия:** Education = TAM entry point. Intelligence Development = SOM differentiation.
