# Тип репозитория

**Тип**: `Pack`

**Source-of-truth**: yes

## Область

**Экосистема развития интеллекта**

Предметная область экосистемы как объекта инженерного описания: что такое экосистема развития, её структура, элементы, характеристики качества, методы оценки.

## Upstream dependencies

- [TserenTserenov/SPF](https://github.com/TserenTserenov/SPF) — Second Principles Framework
- [ailev/FPF](https://github.com/ailev/FPF) — First Principles Framework

## Downstream outputs

- [aisystant/DS-ops](https://github.com/aisystant/DS-ops) — управленческая реализация экосистемы
- Документация экосистемы (surface)

## Non-goals

- НЕ является управлением проектом экосистемы (это DS-ops)
- НЕ является структурой репозиториев (это FMT-S2R)
- НЕ содержит кода, сервисов
- НЕ описывает «как построить экосистему» — только «что это и как оценить»
